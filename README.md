# 🚀 AI-Powered Environmental Sentinel
*Part of the AIRAVAT 3.0 Initiative - IEEE CS S.P.I.T.*

## 📋 Project Overview
This is a Streamlit-based web application designed to monitor environmental data in real-time. The app integrates with **Supabase** for secure data storage and uses Python for backend processing.

## 🛠️ Features
* **Real-time Monitoring:** Track environmental metrics via a clean dashboard.
* **Database Integration:** Securely stores and retrieves logs using Supabase.
* **Cloud Hosted:** Optimized for deployment on Streamlit Community Cloud.
